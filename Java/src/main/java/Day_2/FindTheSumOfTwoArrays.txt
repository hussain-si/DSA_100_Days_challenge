Problem Description
You are given two arrays. Return the sum of all elements in both arrays.


Hint: Use a new variable sum, which can be used to store and return the sum.

Hint: Explore loops to iterate the arrays one by one and add their values to sum.


Sample Input 1
[1,2,3] [4,5,6]

Sample Output 1
21

Explanation 1
We get 1 + 2+ 3 + 4 + 5 + 6 = 21
